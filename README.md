# Subsistence Sandwich Shopping Cart

## Overview
This repository contains an online food ordering website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website simulates a sandwich shop ordering system and demonstrates JavaScript programming concepts including dynamic shopping cart functionality, event handling, DOM manipulation, item quantity tracking, and interactive e-commerce user experiences.

## Features
* Online sandwich ordering system
* Dynamic shopping cart functionality
* Add-to-cart item management
* Automatic quantity tracking for duplicate items
* Interactive menu browsing
* Product catalog display
* Responsive layout using Flexbox
* Customer order management interface

## Technologies Used
* HTML5
* CSS3
* JavaScript
* DOM Manipulation
* Event Handling
* Flexbox

## Project Files
* `sub_menu_txt.html` – Main menu and ordering webpage
* `sub_cart_txt.js` – Shopping cart functionality and item tracking
* `sub_base.css` – Base stylesheet
* `sub_layout.css` – Layout and responsive styling
* Product images and branding assets

## Skills Demonstrated
* Front-end web development

* Shopping cart implementation
* Data tracking and management
* User interface development
